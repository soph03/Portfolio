User Review

Since this application is only available to android user, I was limited by the quantity of research and had to resort to user reviews on the Play Store.

After looking over the reviews, I learned that majority of the users were either unaware that the Google My Maps application existed or that My Maps is fully leaveraged only when it is used with Google Maps. Many users were fustrated about existing features available on the web application but not on the mobile application.

Design Changes

While I did maintain most of the previous design patterns from Google My maps, I made some minor changes to a couple heuristic error that I found in the application. The first one being the popup. The old popup has orphans which can cause uncomfortable reading and visual imbalance to the user. To avoid the problem I have increased the line width.

The second change I made was to the cancel button of the Tags popup. The original cancel button had the same visual design as the save button, however in the new interface the cancel button is visually different. This difference allows the primary action 'save' to have stronger visual weight.
